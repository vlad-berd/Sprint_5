Проект по автоматизации тестирования веб-приложения Stellar Burgers.

👉 Основной фокус: проверка пользовательских сценариев и корректности работы интерфейса.

---

## 📌 О проекте

В проекте реализованы автотесты для проверки полного пользовательского сценария:
- регистрация  
- авторизация  
- работа с личным кабинетом  
- навигация по приложению
- взаимодействие с конструктором  

---

## 🔧 Стек

- Python  
- Pytest  
- Selenium WebDriver  
- Faker  

---

## 🧪 Покрытие тестами

### 👤 Пользователь
- регистрация (валидные и невалидные данные)  
- авторизация  
- переходы на страницы логина  

### 🧑‍💻 Личный кабинет
- переход в профиль  
- выход из аккаунта  
- навигация внутри профиля  

### 🍔 Конструктор
- переходы между разделами  
- работа с элементами интерфейса  

---

## ⚠️ Что проверяется

- корректность пользовательских сценариев  
- обработка ошибок при вводе данных  
- переходы между страницами  
- доступность функциональности для пользователя  

---

## 🧠 Подход к тестированию
- тестирование пользовательских сценариев  
- проверка позитивных и негативных кейсов  
- использование Faker для генерации данных  
- разделение тестов по функциональным модулям  

---
